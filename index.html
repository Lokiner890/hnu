Про мене

Я — веб-розробник, який спеціалізується на створенні інтерфейсів користувача з фокусом на продуктивність, доступність та естетику. Використовую сучасні технології як React, Next.js, Tailwind CSS та TypeScript.

Постійно навчаюся новому, працюю з API, будую SPA, інтегрую бекенд та забезпечую кросбраузерність.

Проєкти

Marvel Explorer

Застосунок на React для перегляду героїв Marvel. Підключення до публічного API, фільтрація, динамічний рендеринг.

Live Chat

Чат з WebSocket на базі Node.js. Реальне оновлення повідомлень, автентифікація, збереження історії.

Особистий блог

Блог на Next.js з Markdown підтримкою. SEO оптимізація, генерація статичних сторінок, інтеграція з GitHub.
